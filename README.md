# بلوقن حماية مايكي - Mikael Security

**المطور:** @DRR_R2  
**القناة:** @phpandpy  
**الإصدار:** 1.0.0

## 📋 الوصف
بلوقن حماية متقدم للمجموعات، يوفر حماية ضد الثغرات والهجمات المختلفة.

## ✨ المميزات
- حماية ضد ثغرات الكروب
- منع التلغيم باستخدام البوت
- قائمة أوامر قابلة للتخصيص من GitHub
- إمكانية إضافة أو تحديث الأوامر بشكل مباشر

## ⚙️ الإعدادات
- **سرعة التنفيذ الموصى بها:**
  - مشرف/مالك مع مميز: 0.3 ثانية
  - مشرف/مالك بدون مميز: 1.2 ثانية
  - ليس مشرف/ليس مالك مع مميز: 0.4 ثانية
  - ليس مشرف/ليس مالك ولا مميز: 4.2 ثانية

## 🔧 التثبيت
1. ضع الملف في مجلد `plugins` في تطبيق ايوجرام أو أي نسخة تدعم البلوقنات
2. فعّل البلوقن من إعدادات التطبيق
3. أعد تشغيل التطبيق

## ⚠️ ملاحظة
المحتوى لأغراض تعليمية فقط. الاستخدام على مسؤوليتك الشخصية.
